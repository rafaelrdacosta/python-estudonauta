Módulo 2 - Estruturas de controle

1) Condições Aninhadas
    if <condição>:
        bloco de instrução
    elif <condição>:
        bloco de instrução
    else:
        bloco instrução

2) Estruturas de repetição for
    #contador - variável criada para iteração de 1 a 10
    for contador in range(1, 10):
        bloco instrução
    Exemplos:
        for contador in range(1, 6):     #a contagem do laço não considera o último número (1 -> 5)
        for contador in range(6, 0, -1): #-1 é para contagem regressiva (6 -> 1)
        for contador in range(0, 7, 2):  #2 será o salto (0 -> 6 de 2 em 2)

3) Estrutura de repetição while
    while <condição>:
        bloco de instrução

    Exemplos:
    contador = 1
    while contador < 10: #irá imprimir de 1 a 9
        print(contador)
        c += 1

    #irá solicitar que seja digitado um número até que o usuário digite 0
    n = 1
    while n != 0:
        n = int(input('Digite um valor: '))

    resposta = 'S'
    while resposta == 'S':
        n = int(input('Digite um valor: '))
        resposta = str(input('Quer continuar? [S/N] ')).upper()
